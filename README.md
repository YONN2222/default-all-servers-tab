# Default All Servers Tab

A plugin for [Pelican Panel](https://pelican.dev) that makes the **All Servers** tab the default view on the servers page instead of **My Servers**.

## Features

- Redirects the servers overview to the `all` tab by default
- Keeps existing `tab` query parameters untouched
- Skips JSON, AJAX, and Livewire requests to avoid side effects

## Installation

### Option 1: Direct URL (Pelican Panel)

Use this URL in the Pelican Panel plugin installer:

```
https://github.com/YONN2222/default-all-servers-tab/releases/latest/download/default-all-servers-tab.zip
```

### Option 2: Upload ZIP (Pelican Panel)

1. Go to the [Releases](https://github.com/YONN2222/default-all-servers-tab/releases) page
2. Download `default-all-servers-tab.zip`
3. Open the Pelican Panel plugin installer
4. Upload the ZIP file there
